## 项目简介

Student Discount 是一个专门为学生整理全球各类学生优惠信息与开通教程的导航站，帮助学生以更低价格使用 Spotify、Adobe、Apple、Microsoft 等品牌的产品和服务。[网站支持多语言，并覆盖 US、UK、SG、IN 等多个国家和地区的优惠政策与价格信息。

## 主要功能

- 汇总各大品牌学生优惠的折扣力度、适用地区和订阅价格，方便快速横向对比。
- 提供从注册、验证到续期、退款的完整开通流程指南，并尽量附带官方来源链接，保证信息可核验。
- 针对常见的学生身份验证失败、地区限制等问题，提供集中排查与解决建议的故障排查中心。
- 维护学生福利与社区栏目，定期更新教育工具、学习相关服务以及共享福利等优惠信息。

## 适合谁使用

- 想要用更低价格订阅 Spotify、Apple Music、Hulu、Disney Plus 等流媒体服务的学生用户。
- 需要购买 Adobe Creative Cloud、Microsoft 365 等学习/创作软件并希望节省预算的学生。
- 想系统了解不同国家/地区学生优惠差异、验证方式和注意事项的留学生或准备出国的学生。

## 如何开始

- 访问 https://studentdiscount.io 获取英文版学生优惠导航与操作说明。
- 访问 https://studentdiscount.io/zh 浏览中文版的品牌列表与详细开通教程，按品牌或国家筛选所需优惠信息。
